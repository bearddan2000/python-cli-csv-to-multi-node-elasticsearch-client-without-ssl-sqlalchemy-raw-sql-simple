# python-cli-csv-to-multi-node-elasticsearch-client-without-ssl-sqlalchemy-raw-sql-simple

## Description
Reads a csv file into a multi node cluster for data in `dog-demo` document.

Uses sqlalchemy to connect and executes raw sql.

## Tech stack
- python
    - flask
    - sqlalchemy
    - elasticsearch
    - elasticsearch_dbapi
- elasticsearch
- kibana

## Docker stack
- python
- elasticsearch
- kibana

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
